Présentation du projet

Intitulé L'Atelier Douceur
Type Site web de partage de recettes
Contexte Projet personnel réalisé dans le cadre du BTS SIO SLAM
Période 2025 - 2026
Rôle Développeur unique (conception et réalisation)

Description

L'Atelier Douceur est un site web communautaire dédié au partage de recettes de desserts. Les utilisateurs peuvent créer un compte, publier leurs propres recettes avec photos et instructions, les modifier ou les supprimer, et commenter les recettes des autres membres.

Fonctionnalités

  • Inscription et connexion utilisateur (authentification sécurisée)
  • Création, modification et suppression de recettes (CRUD)
  • Upload de photos pour illustrer les recettes
  • Système de commentaires sur les recettes
  • Recherche de recettes par nom ou catégorie
  • Page profil utilisateur avec liste de ses recettes

Technologies utilisées

PHP MySQL Bootstrap 5 HTML/CSS
Langage back-end PHP (sans framework)
Base de données MySQL
Front-end HTML, CSS, Bootstrap 5
Serveur local XAMPP / WampServer

Base de données

Principales tables :

  • utilisateurs : id, nom, email, mot_de_passe, date_inscription
  • recettes : id, titre, description, ingredients, etapes, photo, id_utilisateur, date_creation
  • commentaires : id, contenu, id_recette, id_utilisateur, date
  • categories : id, nom
