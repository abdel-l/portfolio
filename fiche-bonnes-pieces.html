Présentation du projet

Intitulé Les Bonnes Pièces
Type Site e-commerce de pièces automobiles
Contexte Projet personnel réalisé dans le cadre du BTS SIO SLAM
Période 2025 - 2026
Rôle Développeur unique (conception et réalisation)

Description

Les Bonnes Pièces est un site e-commerce spécialisé dans la vente de pièces automobiles. Il permet aux clients de parcourir un catalogue de produits, d'ajouter des articles à un panier, et de passer commande après authentification. Un espace d'administration permet de gérer les produits, les stocks et les commandes.

Fonctionnalités

  • Catalogue de produits avec filtres par catégorie
  • Fiche produit détaillée (photo, description, prix, stock)
  • Système de panier (ajout, modification, suppression d'articles)
  • Inscription et connexion utilisateur
  • Passage de commande avec récapitulatif
  • Espace administration : gestion des produits (CRUD), gestion des commandes

Technologies utilisées

PHP MySQL Bootstrap 5 HTML/CSS
Langage back-end PHP (sans framework)
Base de données MySQL
Front-end HTML, CSS, Bootstrap 5
Serveur local XAMPP / WampServer

Base de données

Principales tables :

  • utilisateurs : id, nom, email, mot_de_passe, role (client/admin)
  • produits : id, nom, description, prix, stock, photo, id_categorie
  • categories : id, nom
  • commandes : id, id_utilisateur, date, statut, total
  • lignes_commande : id, id_commande, id_produit, quantite, prix_unitaire
