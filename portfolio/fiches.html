E5 — Projet 2

Les Bonnes Pièces

Développement applicatif

Contexte

Projet simulant un site e-commerce de vente de pièces automobiles. Maîtrise de la gestion d'un catalogue produits, d'un système de panier et d'une interface d'administration.

Environnement technique

  • Serveur : Apache (XAMPP)
  • Backend : PHP 8
  • Base de données : MySQL / PhpMyAdmin
  • Frontend : HTML5, CSS3, Bootstrap 5, JavaScript
  • Versioning : Git / GitHub

Compétences mobilisées

CodeCompétenceMise en œuvre
B2.1Conception et développement d'une solution applicativeArchitecture MVC simplifiée, panier en session, interface admin complète
B2.2Maintenance corrective ou évolutiveAjout de fonctionnalités (filtres produits, gestion des stocks)
B2.3Gestion des donnéesBDD relationnelle (produits, catégories, utilisateurs), requêtes PDO
B2.4Travail en mode projetDécoupage en sprints, versioning GitHub, cahier des charges
B2.5Production de documentationManuel utilisateur, documentation BDD, MLD

Difficultés & solutions

  • Panier multi-produits : Persistance en session — tableau associatif $_SESSION
  • Droits admin : Champ role en BDD avec vérification à chaque page protégée
  • Performances : Optimisation des jointures SQL pour le catalogue avec filtres
